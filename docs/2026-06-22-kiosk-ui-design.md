# 면상 판독기 — 셀프 키오스크 UI 재설계 (2026-06-22)

## 목표
행사장에서 참가자가 스스로 반복 사용. 진행요원 개입 최소화.

## 화면 (전체화면)
- 캔버스 1280×720 고정, 창은 전체화면으로 확대 (마우스 좌표는 캔버스 기준)
- 왼쪽 960×720: 카메라 영상 (좌우반전)
- 오른쪽 320px: 흰 배경 사이드바. 상단 제목+상태, 하단 파란 버튼 세로 정렬

## 상태 흐름 (자동 루프)
1. SETUP  인원수 설정 — `[−] N명 [+]`, `[시작]`
2. CAPTURE 촬영 대기(idx/total) — `[촬영]` → 3·2·1 카운트다운 → 플래시 → 측정
   - 얼굴 미검출: 경고 후 같은 번호 재촬영
3. RESULT 개인 결과 — 촬영본 정지화면 + 닮은 사진/닮음도, `[다음]`
4. FINAL  우승 발표 — 우승자 정지화면, `[새 게임]` → SETUP

## 조작
- 참가자: 모든 버튼 마우스/터치 (cv2.setMouseCallback + 사각형 히트테스트)
- 관리자: 키보드로 `ADMIN_EXIT_CODE`("quit") 타이핑 시 종료. 참가자 오종료 방지
- 창 X 닫기도 안전 종료

## 연출
- 카메라 좌우반전, 촬영 카운트다운, 기존 플래시 유지
- 흰 배경 / 파란 버튼 통일 / 흰 글자

## 코드
- 검출·모델·매칭·한글IO 함수 유지
- UI 레이어 교체: 레이아웃 상수, draw_round_rect, Button, put_texts_kr(1패스), compose_canvas, countdown, 상태머신 main()
- 임시저장 덮어쓰기 유지
