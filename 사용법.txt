면상 판독기 (셀프 키오스크) 사용법
====================================

[준비]
1. '비교군' 폴더에 닮음 비교 대상 얼굴 사진을 넣으세요.
   - 파일 이름이 곧 결과에 표시되는 '사진 이름' 입니다. (예: 강아지상.jpg, 김연예인.png)
   - 한 사진에 얼굴 하나가 잘 보이게.
2. '임시저장' 폴더 : 촬영한 사진이 person_01.jpg ... 형태로 자동 저장됩니다.
3. 'models' 폴더 : 얼굴 검출/인식 모델(ONNX). 그대로 두세요.

[실행]
  python 면상_판독기.py
  → 전체화면으로 열립니다. (참가자가 직접 조작)

[참가자 사용 순서 — 모두 화면의 파란 버튼을 마우스/터치로]
1. [−] [+] 로 인원수를 정하고 → [시작]
2. [촬영] → 3·2·1 카운트다운 후 플래시와 함께 촬영
   - 얼굴이 안 잡히면 같은 번호로 다시 촬영합니다.
3. 닮은 사진 이름 + 닮음도(%) 표시 → [다음]
4. 인원수만큼 2~3 반복
5. 전체 중 가장 닮은 사람 발표 → [새 게임] 누르면 처음부터 다시
   (프로그램 재실행 없이 다음 팀 바로 진행)

[진행요원(관리자)]
- 종료: 키보드로 'quit' 를 차례로 입력 (참가자 오종료 방지)
- 창의 X 버튼으로 닫아도 종료됩니다.

[설정 바꾸기 — 면상_판독기.py 상단 '설정' 구간]
- ADMIN_EXIT_CODE : 종료 암호 글자 (기본 "quit")
- FULLSCREEN      : 전체화면 on/off
- MIRROR          : 카메라 좌우반전 on/off
- EXAGGERATE_FULL : 닮음도 과장 정도 (작을수록 더 과장)
- 색상/글자크기/카운트다운 시간 등도 같은 구간에서 조정

[참고]
- 닮음도 = SFace 임베딩 코사인 유사도(%) (값이 클수록 더 닮음).
